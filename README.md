# Cognizant Deepskilling - Hands-On & Extra Work

This repository contains my solutions, hands-on exercises, practice programs, and additional work completed as part of the Cognizant Deepskilling Program.

## Repository Structure

The repository is organized into multiple folders based on modules, topics, assignments, and practice activities.

Each folder contains:

* Source code files
* Supporting resources (if applicable)
* A dedicated instruction document explaining:

  * The purpose of the task
  * Steps to execute the code
  * Expected outputs (where applicable)

## How to Use This Repository

1. Navigate to the folder corresponding to the topic or assignment you wish to explore.
2. Read the instructions provided within that folder before executing any files.
3. Follow the setup and execution steps mentioned in the folder-specific documentation.
4. Run the programs as described in the respective folder instructions.

## Notes

* Each folder is intended to be self-contained.
* Folder-level documentation should always be referred to first for execution details and project-specific requirements.
* Solutions are provided for learning, practice, and reference purposes.
